COMO FUNCIONA A INTERNET?

A internet é baseada em um conceito antigo, conhecido na computação como arquitetura cliente-servidor. Nela, existem varios computadores interligados a uma rede, e esta, interligada a outra, formando assim, um aglomerado de computadores interligados, transferindo e recebendo informações.

































O QUE É UM SERVIDOR WEB?

De maneira simplificada, um servidor web é um computador responsável pelo armazenamento, processamento e entrega dos arquivos dos sites para os navegadores.

































World Wide Web

A World Wide Web (conhecida como "WWW", "Web" ou "W3") é o universo de informações acessíveis pela rede, a personificação do conhecimento humano.
